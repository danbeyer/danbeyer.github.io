Project Description

This website allows users to search public .CSV files from the federal Medicare Electronic Health Record (EHR) Incentive Program to find the EHR systems in use by healthcare providers. An AJAX XHR object utilizes the National Provider Registry API is utilized to find the NPI numbers for the healthcare providers specified by the users' search criteria. The site then uses the wonderful Papaparse to parse the .CSV files into JSON format. The JSON object is then searched to find the EHR systems in use by the corresponding NPI numbers, and a dynamic table is created to display the results.

The site is a work in progress. Future improvements/plans:

  • To bypass CORS issues and directly access the .CSV file from the healthit website, consider using node.js or some other form of server to server communication
  • Allow user to specify type of provider to narrow search results
  • Add Loader loading animation during searching/parsing
